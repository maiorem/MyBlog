# MyBlog
Spring Boot, JPA 활용한 개인 사이트 만들기      

<hr />

## 사용 기술
* VIEW : Thymeleaf         
* SERVER : JAVA8    
* DATABASE : MySQL, JPA    
* WAS : Tomcat8    
* 배포 : AWS    
* Framework : Spring Boot, Hibernate
* Build : Maven     
* API : Rest-API, 카카오 로그인, FIREBASE, Summernote 이후 추가
